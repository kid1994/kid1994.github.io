回忆我的大学

Posted by 剑知 on 2019-10-27
剑知

  • 我的梦呓

我这人一直是后知后觉，反应比较慢，发生了什么事当时不会有多少感受，随着时间一点点地过去才会慢慢有感觉。

不知不觉已经毕业四个多月了，真的感觉时间过得很快。其实毕业的时候，包括参加毕业典礼、吃散伙饭的时候，自己并没有感觉到多大的不舍和留恋，反倒是现在那种怀念的感觉变得愈发强烈。可能就是需要你去逐渐意识到一件东西不再存在的时候，才会开始想念吧。

在研三上学期的时候，那时候已经早早地定了工作，暂时又不想做毕设，就开始想着要毕业的事了。也许是为了要有点仪式感，我提前半年就准备着如何和学校分别。每天晚上就在校园以及周边到处溜达，拍了很多照片和视频，想着要剪成一个视频，但最后还是因为太懒没付诸实现。又想到到时毕业时大家肯定会在朋友圈发一些感言，那我应该发点什么内容呢？于是早早地开始构思，并且在日记里打下了草稿，今天又翻了一下，草稿的内容如下：

再见了，我的操场，麻烦继续帮我守住我的心事和秘密
再见了，我的图书馆
再见了，西南门的煎饼果子，嗯，对，不要辣
再见了，后街的小吃，手抓饼的老大爷
再见了，404宿舍，睡了你这么多年，受累了
// 再见了，我后青春时代里的那个女生
再见了，西电
若干年以后再见时，再告诉我这七年间的成长
// 到临了也没能带走个对象陪我去杭州

一如既往地矫情:)。但最后真正等到毕业的时候却又什么都没发，连毕业照都没发。

虽然研究生期间也有一些让我感到“痛苦”的回忆，但相较于本科而言，美好的回忆还是多了不少。由于研究生有了实验室，因此相对整个大班级而言，实验室的同学能够走得更近，让我有了点“同学情”的感觉，现在毕业后也一直保持着联系，希望能保持下去。

但研究生期间没变的是，我一直保持着独立，喜欢有一个自己的空间，并且不欢迎别人走进来。大多数时候，我还是更喜欢一个人呆着，一个人吃饭，一个人去图书馆学习，一个人晚上等操场熄灯后去打球，一个人默默品尝自己的喜怒哀乐。独处让我感到放松，有安全感。但比本科还是好得多了，和实验室同学们一起玩的时候，我可以真正地感受到快乐，可以笑得很开心。

当然了，要说有没有遗憾，那肯定还是有的。其实从一开始我就想在校园里谈一场恋爱，并且憧憬着能从校服走到婚纱，但现在想想可能是一种奢望吧。我有尝试过，努力过，但并没有取得自己想要的结果。究其原因，我觉得还是自己不够自信，并且方法和方向有误。

我知道，这种感觉会越来越强烈，等到一个迸发的时间点，就该回去看看了。

骆家庄西苑三区。
