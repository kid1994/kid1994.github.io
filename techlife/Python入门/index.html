Python的定位

我已经会了C/C++/Java等语言，为什么还需要再学习Python呢？

首先，和C/C++/Java相比，Python的优势在于轻便。C/C++/Java更适用于一些大型项目，需要经历编译、测试、部署、修改、再编译测试部署等阶段，而作为一门动态的脚本型语言，Python更适用于一些中小型项目，且跨平台，能够快速地完成任务。

完成同样的任务，所需的Python代码比C/C++/Java代码更短，原因在于：

  • Python中高级的数据结构能够更简洁地表达数据的操作
  • 使用缩进，而不是中括号来表示语句块
  • 不需要对变量进行声明

另外，和Shell脚本相比，Python是一门真正的编程语言，它比Shell脚本多了更多的数据结构支持，还有广泛的标准库支持，而Shell脚本主要擅长文件及文本的操作。

函数式编程工具

Python中有3个内置的函数，可以很方便地支持函数式编程：

  • filter(function, sequence):过滤掉sequence中function(item)为false的元素。如果sequence的类型是str、unicode、tuple，则返回的结果类型不变；否则返回的结构类型都是list。
  • map(function,sequence):对squence中的每个元素都调用一遍function，返回结果的list。
  • reduce(function, squence):首先对squence中的前两个元素调用function，然后再对结果和第3个元素调用function，以此类推，最后返回最终的结果。

模块

为了复用和扩展，Python支持以模块化的方式管理脚本文件。一个模块就是一个包含一系列定义和语句的脚本文件，通过import关键字导入模块，来达到对模块的复用。

模块中的语句通常用来对模块进行初始化，只有在首次import模块，或者直接执行模块脚本的时候才会执行这些语句。

模块的搜索路径

面向对象编程

Python的类机制混合了C++和Modula-3的类机制，并在此基础上增加了新的语法和语义。

作用域和命名空间

命名空间是一组对象的集合，可以想象为一个字典，键是对象的名称，值是对象。以下都属于命名空间：

  • 内置的函数、异常对象
  • 一个模块中所有的全局变量
  • 一个函数中所有的局部变量
  • 一个对象的所有属性

任意两个命名空间中的对象之间没有任何关联，从而避免冲突。

作用域是指一块代码区域，在这个区域中，可以直接访问一个命名空间中的对象。

在Python中，函数对象和方法对象不是同一个概念：

  • classname.method()是函数对象
  • objectname.method()是方法对象

方法的第一个参数总是self，即对象本身。x是一个对象，以下两种调用方式是等价的：

  • x.method()
  • class.method(x)

底层的实现原理是：方法对象 = 对象实例 + 函数对象。

Python中不存在私有的实例变量，但是一个通用的惯例是：以_开头的属性都是非公有的。
