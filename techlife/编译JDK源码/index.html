前期准备

下载openjdk源码

首先是关于要下载哪个版本的jdk源码，在Mac os X 13上，推荐下载openjdk 9u版本。不过最终我是在Ubuntu 14下构建的，选择的是openjdk 7版本。

然后有如下三种下载源码的方法：

  • 通过mercurial下载安装
  • 自己手动下载压缩包
  • 从github上下载别人上传的源码

最推荐的还是第一种方法，但由于网速原因，才不得以使用后两种方法。

构建过程中遇到的问题

error 1 gcc compiler is required

解决方法：这个问题是由于XCode使用的是clang编译器，不支持gcc，而编译JDK默认需要gcc编译器。

但是本版本的clang已支持大部分gcc，所以可以就使用clang，但是需要把common/autoconf/generated-configure.sh文件的第20061和21640行注释掉；当然最好的方法还是安装gcc编译器:

error 2 could not find freetype

解决方法：通过安装XQuartz来安装freetype,然后执行configure时加上选项--with-freetype-include=/opt/X11/include/freetype2/ --with-freetype-lib=/opt/X11/lib

error 3 ordered comparison between pointer and zero

解决方法：打开hotspot目录中的

  • src/share/vm/memory/virtualspace.cpp 将其中if (base() > 0) {改为if (base() != NULL) {
  • src/share/vm/opto/lcm.cpp 将其中if (Universe::narrow_oop_base() > 0) {改为 if (Universe::narrow_oop_base() != NULL) {
  • src/share/vm/opto/loopPredicate.cpp 将其中assert(rng->Opcode() == Op_LoadRange || _igvn.type(rng)->is_int() >= 0, "must be"); 改为 assert(rng->Opcode() == Op_LoadRange || iff->is_RangeCheck() || _igvn.type(rng)->is_int()->_lo >= 0, "must be");

error 4 a fatal error has been detected by the java runtime environment

error 5 cannot find source for project jaxp

解决方法：添加环境变量export ALLOW_DOWNLOADS=true

error 6 protocol switch unsafe, not allowed

解决方法：将以下三个文件下载后置于OpenJDK解压后根目录下的drop目录下，

error 7 this os is not supported

解决方法：修改 hotspot/make/linux/Makefile，找到 SUPPORTED_OS_VERSION变量定义的地方，在后面追加 3%

##error 8 cc1plus all warnings being treated as errors

解决方法：搜索文件hotspot/make/linux/makefiles/gcc.make中的-Werror,注释掉即可。

error 9 gamma relocation error

解决方法：修改文件hotspot/make/linux/Makefile,注释掉所有test_gamma。

error 10 unrecognized command line option -mimpure-text

解决方法：去掉文件jdk/make/common/shared/Compiler-gcc.gmk中的-mimpure-text。

error 11 time is more than 10 years from present

解决方法：修改jdk/src/share/classes/java/util/CurrencyData.properties文件，把其中所有时间都改为10年内。

error 12 collect2 error ld returned 1 exit

解决方法：在openjdk/jdk/make/javax/sound/jsoundalsa/Makefile里面
刪除 LDFLAGS += -lasound
加入 OTHER_LDLIBS += -lasound
